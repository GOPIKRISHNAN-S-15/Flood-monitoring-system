The system continuously monitors water levels and automatically responds to flood threats, ensuring community safety and property protection.

How It Works

Understanding the flood detection and mitigation process

1
Water Detection

Ultrasonic sensor continuously measures distance between sensor and water surface, providing real-time water level data.

2
Data Transmission

NodeMCU ESP8266 processes sensor data and transmits it to the cloud server via WiFi connection.

3
Intelligent Analysis

Server analyzes water level data against predefined thresholds and triggers appropriate alerts and responses.

4
Automated Response

System automatically activates flood gates, triggers alarms, and sends notifications when flood conditions are detected.

Hardware Components

Detailed breakdown of system components

NodeMCU ESP8266

Function: Main microcontroller unit

Features:

  • WiFi connectivity
  • GPIO pins for sensors
  • Low power consumption
  • Arduino IDE compatible
Ultrasonic Sensor (HC-SR04)

Function: Distance measurement

Features:

  • Range: 2cm to 400cm
  • ±3mm accuracy
  • 40kHz ultrasonic frequency
  • Low power operation
Servo Motor (SG90)

Function: Flood gate control

Features:

  • 180° rotation angle
  • Digital servo control
  • High precision positioning
  • Reliable operation
Buzzer

Function: Local alert system

Features:

  • High decibel output
  • Low power consumption
  • Instant activation
  • Weather resistant
16x2 LCD Display

Function: Local data display

Features:

  • Real-time readings
  • Clear visibility
  • Backlight support
  • I2C interface

Water Level Detection Logic

How the system determines flood risk levels

Normal Level

>80 cm

Distance from sensor

Actions:

  • Green status indicator
  • Normal monitoring mode
  • Flood gate remains closed

Warning Level

50-80 cm

Distance from sensor

Actions:

  • Yellow alert status
  • Warning notifications
  • Increased monitoring

Danger Level

<50 cm

Distance from sensor

Actions:

  • Red danger alert
  • Buzzer activation
  • Auto gate opening
  • Emergency notifications
